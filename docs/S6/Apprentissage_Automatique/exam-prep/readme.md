# Exam Prep -- Apprentissage Automatique

## Analyse des 39 annales (2013-2023)

Les examens sont generalement donnes par deux enseignants (Christian Raymond et Vincent Claveau) avec des styles complementaires. L'examen dure 2h a 2h30 et comporte 3 a 5 exercices.

---

## Structure typique de l'examen

| Partie | Contenu | Points estimes |
|--------|---------|---------------|
| **Questions de cours** | Definitions, principes, comparaisons | 20-30% |
| **Exercice Arbres** | Calcul d'entropie, construction, elagage | 25-30% |
| **Exercice Bayes/KNN** | Theoreme de Bayes, classification NB, KNN | 20-25% |
| **Questions TP** | Code sklearn, interpretation, CCP | 15-20% |
| **Bonus/Avance** | Boosting, inference grammaticale | 5-10% |

## Strategie d'examen

### Gestion du temps (2h)

| Etape | Temps |
|-------|-------|
| Lecture complete du sujet | 10 min |
| Questions de cours | 20 min |
| Exercice principal (arbres/Bayes) | 40-50 min |
| Exercice secondaire | 25-30 min |
| Questions TP | 15-20 min |
| Relecture | 5 min |

### Conseils prioritaires

1. **Commencer par les questions de cours** : points faciles et rapides.
2. **Poser les formules avant de calculer** : les correcteurs donnent des points pour la demarche meme si le resultat numerique est faux.
3. **Dessiner les arbres etape par etape** : meme si c'est long, c'est ce qui rapporte des points.
4. **Ne pas oublier P(X) dans Bayes** : erreur tres frequente.
5. **Passer si bloque** : les exercices sont generalement independants.
