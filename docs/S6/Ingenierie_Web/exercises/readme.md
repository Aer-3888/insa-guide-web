# Exercices -- Ingenierie Web

## Organisation

Guides detailles pas a pas pour les 5 TPs du cours d'Ingenierie Web, couvrant les 377 fichiers source du dossier TP. Chaque guide contient la structure du projet, le code complet commente, les commandes curl de test, et la progression etape par etape.

| Fichier | TP | Theme | Fichiers source couverts |
|---------|-----|-------|-------------------------|
| [tp1_javascript.md](/S6/Ingenierie_Web/exercises/tp1-javascript) | TP1 | JavaScript, DOM, Canvas, ES6, Jest | 18 fichiers (HTML, JS, CSS, tests) |
| [tp2_rest_api.md](/S6/Ingenierie_Web/exercises/tp2-rest-api) | TP2 | REST API Jersey/JAX-RS, Swagger, JAXB | 36 fichiers (Java, HTML, JS, XML) |
| [tp3_data_formats.md](/S6/Ingenierie_Web/exercises/tp3-data-formats) | TP3 | XML, DTD, XSD, Relax NG, JSON | 14 fichiers (XML, JSON, Java, JS) |
| [tp4_orm_jpa.md](/S6/Ingenierie_Web/exercises/tp4-orm-jpa) | TP4 | JPA, Hibernate, entites, relations | 13 fichiers (Java, XML) |
| [tp5_spring_boot.md](/S6/Ingenierie_Web/exercises/tp5-spring-boot) | TP5 | Angular, Spring Boot, services, securite | 45+ fichiers (TypeScript, Java) |

## Contenu de chaque guide

- **Structure du projet** : arborescence complete des fichiers
- **Code source complet** : chaque fichier cle reproduit et commente
- **Documentation de l'API** : points d'acces REST avec methodes, parametres, codes de retour
- **Tests curl** : commandes pour tester chaque point d'acces
- **Evolution des fichiers** : quels fichiers changent a chaque etape du TP
- **Concepts essentiels** : tableaux recapitulatifs des annotations et motifs de conception
