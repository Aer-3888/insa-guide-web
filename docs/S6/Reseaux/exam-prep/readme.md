## Analyse des annales (2018-2024)

### Themes recurrents par frequence

| Theme | Frequence | Annees |
|-------|-----------|--------|
| Calculs de sous-reseaux (masque, CIDR, VLSM) | Chaque annee | 2018-2024 |
| Routage pas a pas (MAC/IP a chaque saut) | Chaque annee | 2018-2024 |
| TCP : handshake, seq numbers, ACK | Frequent | 2019, 2021, 2022, 2024 |
| Analyse de captures (Wireshark) | Frequent | 2020, 2021, 2022, 2024 |
| ARP (request/reply, broadcast/unicast) | Frequent | 2018, 2019, 2021, 2024 |
| Fragmentation IP | Regulier | 2019, 2022, 2024 |
| Programmation socket (C) | Regulier | 2020, 2022, 2024 |
| DNS, HTTP, DHCP | Regulier | 2018, 2021, 2024 |
| Comparaison TCP vs UDP | Regulier | 2019, 2021, 2022 |
| RIP vs OSPF | Occasionnel | 2018, 2022 |
| VLAN | Occasionnel | 2021 |

### Format typique du DS

- **Duree** : 2 heures, sans documents.
- **3-4 exercices** independants, chacun sur un theme.
- **Exercice 1** (typiquement) : sous-reseaux/VLSM.
- **Exercice 2** : routage pas a pas ou analyse de capture.
- **Exercice 3** : TCP/UDP (handshake, seq numbers) ou programmation socket.
- **Exercice 4** : question ouverte ou protocole applicatif.
